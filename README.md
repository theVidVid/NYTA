# NYTA Admin Panel

NYTA is a fictional admin panel for a Tech bootcamp that allows an administrative user the ability to create/edit/delete Courses, Cohorts, Students, and Instructors.

# Built with:

* Ruby-on-Rails

* PostgreSQL
